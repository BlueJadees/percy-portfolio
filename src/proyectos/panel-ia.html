Panel IA

Una plataforma web propia construida sobre ComfyUI para generar imágenes y video con IA, con gestión de modelos, galería y utilidades de flujo de trabajo. Uno de los proyectos a los que más tiempo le he dedicado.

Panel IA envuelve ComfyUI en una aplicación web hecha en Flask, transformando un motor potente pero técnico en una herramienta de trabajo cómoda, con todo el flujo de generación integrado en una sola interfaz autoalojada.

El foco estuvo en agilizar el flujo de trabajo de la generación de imágenes. La pieza central es una bóveda de LoRAs (modelos de ajuste fino): la plataforma incluye una herramienta propia para descargar y subir LoRAs directamente desde CivitAI a esa bóveda, resolviendo la gestión de modelos sin salir de la aplicación.

Alrededor de eso construí un ecosistema de utilidades: un sistema de galería para las generaciones, creación de video con Wan2.1-I2V, integración con la API de Danbooru, traducción de prompts vía Google Translate, y herramientas para mejorar la calidad de las imágenes.

Más allá de lo personal, usé Panel IA como banco de pruebas de concepto: exploración visual de personajes e imágenes para mis videojuegos y para conceptos de empresa — un entorno propio para prototipar arte rápido antes de comprometer tiempo de producción.

Capacidades

Bóveda de LoRAs + CivitAI

Gestión de modelos LoRA con descarga y subida directa desde CivitAI a una bóveda local, integrada en la propia plataforma.

Imagen y video

Generación de imágenes sobre ComfyUI y de video mediante Wan2.1-I2V, con un sistema de galería para organizar los resultados.

Utilidades de prompt

Integración con la API de Danbooru, traducción de prompts vía Google Translate y herramientas para mejorar la calidad de las imágenes.

Stack & tecnologías

Python Flask ComfyUI Wan2.1-I2V (video) API CivitAI API Danbooru Google Translate API Docker (HomeLab)
